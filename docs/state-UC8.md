- **Cross-fibre persistence (UC8 Defn §1.1).** $h(S):=|\{R\in\mathcal R_W^{\mathrm{lg}}:S\in\mathcal D(R)\}|$ for $S\subsetneq W$, $S\ne\emptyset$. Lemma 1.1: $R\subseteq R'$ + $S\in\mathcal D(R')$ ⇒ $S\in\mathcal D(R)$ on common domain (by UC6 Cor 3.2); the set $\{R:S\in\mathcal D(R)\}$ is a down-ideal in $(\mathcal R_W^{\mathrm{lg}},\subseteq)$ on the common $\mathcal S(R)^\uparrow$-domain.
- **Chain-cumulative identity (UC8 Thm 1.1).** $D^{\mathrm{lg}}(W)=\sum_{S\subsetneq W,S\ne\emptyset}h(S)\phi(S)$. The Phase-1 residual is a single persistence-weighted sum over sub-traces.
- **Cross-fibre 1-rich class at $k=3$ (UC8 Thm 1.2).** $\sum_{|S|=1}h(S)\ge\sum_{|S|=2}h(S)\Rightarrow D^{\mathrm{lg}}(W)\le 0\Rightarrow$ Frankl. Properly contains UC7 Thm 1.3 (per-fibre 1-rich ⇒ cross-fibre 1-rich, reverse fails — Example 1.3).
- **Cross-fibre half-level-deficient class at $k\ge 4$ (UC8 Thm 1.4).** $\sum_{|S|\le k/2}h(S)\ge\sum_{|S|>k/2}h(S)$ (equivalently $\sum_Sh(S)\phi(S)\le 0$) $\Rightarrow$ Frankl. Properly contains UC7 Thm 1.2.
- **Cumulative-incidence identity (UC8 Lemma 2.1).** $\sum_W\sigma_{\mathrm{rest}}(W)=2\sum_{x\in U_{\mathcal W}}\deg_{\mathcal F\setminus\mathcal W}(x)\deg_{\mathcal W}(x)-kN|\mathcal F\setminus\mathcal W|$. The Phase-2 lever rewrites as a single inner-product condition.
- **Cross-incidence cumulative-balanced class (UC8 Thm 2.2).** $\sum_x\deg_{\mathcal F\setminus\mathcal W}(x)\deg_{\mathcal W}(x)\ge(kN/2)|\mathcal F\setminus\mathcal W|\Rightarrow\sum_W\sigma_{\mathrm{rest}}(W)\ge 0$. - **Chebyshev coupling (UC8 Lemma 2.3, Thm 2.4).** Similarly-ordered $\deg_{\mathcal F\setminus\mathcal W},\deg_{\mathcal W}$ ⇒ $\langle\deg_{\mathcal F\setminus\mathcal W},\deg_{\mathcal W}\rangle\ge\frac{1}{|U_{\mathcal W}|}\sum\deg_{\mathcal F\setminus\mathcal W}\cdot\sum\deg_{\mathcal W}$ ⇒ (under $|U_{\mathcal W}|\le 2k$, UC5 Thm 3.4) Frankl *without* $\Gamma$-sparse.
- **Domain-mismatch bound (UC8 Lemma 3.1).** $|\Delta_{\mathrm{dom}}(R,R')|\le|\mathcal S(R)^\uparrow\triangle\mathcal S(R')^\uparrow|$.
- **Cauchy–Schwarz cross-deficiency-overlap bound (UC8 Thm 3.2).** $|\Delta_{\mathrm{cross}}(R,R')|\le\sum_{T\in\mathcal C}|\delta_R(T)\cap\delta_{R'}(T)|\le k\sqrt{\mathrm{Def}(R)\mathrm{Def}(R')}$ (using $|\delta|\le k$ on $W$).
- **F-iii square-defect bound (UC8 Thm 3.1).** $|X(R,R')|\le|\mathcal S(R)^\uparrow\triangle\mathcal S(R')^\uparrow|+k\sqrt{\mathrm{Def}(R)\mathrm{Def}(R')}$. The Lever-3 ask discharged: $f(|\mathcal S(R)|,|\mathcal S(R')|,|\delta_R\cap\delta_{R'}|)$ = up-closure symmetric difference + geometric mean of deficiency masses with prefactor $k$.
- **Cumulative excess bound (UC8 Cor 3.3).** $\sum_{R,R'}|X(R,R')|\le\sum_{R,R'}|\mathcal S(R)^\uparrow\triangle\mathcal S(R')^\uparrow|+k|\mathcal R_W|\sum_R\mathrm{Def}(R)$.
- **Extended fully-signed class (UC8 Thm 4.1).** Intersection of cross-fibre 1-rich (or half-level-deficient) ∩ cross-incidence cumulative-balanced (+ $\Gamma$-sparse OR similarly-ordered + concentrated) ∩ F-iii-bounded excess (+ §1 at $R^*$) ⇒ Frankl unconditionally.
